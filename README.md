# Cinematic

Welcome to Cinematic, a platform that provides movie lovers with an immersive experience to discover, share, and indulge in their favorite movies, while keeping their friends in the loop about the latest and greatest film recommendations in the industry.

## Features

Discover the latest and greatest film recommendations
Share your own movie suggestions with others
Like and comment on your favorite movies
Follow friends to see their movie recommendations
Search for movies by title, genre, or keywords
Create and join groups based on shared movie interests

## Getting Started

To use Cinematic, simply open the website and start exploring! To get all of the features you have to create an accout. You can search for movies, and add your favorite movies to your profile. You can send movie suggestions to your friends and follow your friends to see their recommendations.

To contribute to the Cinematic community, you can submit a new movie, share your own movie suggestions

## Getting Started with Development

To run Cinematic on your local machine, follow these steps:

### Clone the repository to your local machine:

```bash
git clone https://github.com/sumanbiswas7/cinematic.git
cd cinematic
```

### Install dependencies:

```bash
npm install
```

Create a .env file in the root directory of the project, and fill in the environment variables according to the env.example file provided. This will be used to set up the database and other important configurations.

### Start the development server:

```bash
npm run dev
```

Visit http://localhost:3000 in your web browser to view the application.

That's it! You should now be able to use Cinematic on your local machine. If you encounter any issues or have any questions, please feel free to reach out to me.
